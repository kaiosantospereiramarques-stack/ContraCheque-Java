 ContraCheque-Java

Este é um programa desenvolvido em **Java** para automatizar o cálculo do salário líquido, considerando as tabelas vigentes de **INSS** e **IRRF**.

O projeto foi criado como parte dos meus estudos de lógica de programação, aplicando conceitos de estruturas condicionais complexas e entrada de dados via console.

# Funcionalidades

- Cálculo automático do desconto **INSS** baseado em faixas salariais.
- Cálculo do **IRRF** (Imposto de Renda Retido na Fonte).
- Consideração de deduções e reduções mensais conforme regras fiscais.
- Exibição detalhada de cada desconto e do valor final a receber.

# Tecnologias Utilizadas

- **Linguagem:** Java
- **Entrada de Dados:** `java.util.Scanner`
- **Lógica:** Estruturas condicionais (`if`, `else if`) e operadores matemáticos.

# Como rodar o projeto

1. Tenha o Java (JDK) instalado em sua máquina.
2. Clone o repositório:
   ```bash
   git clone https://github.com
   ```
3. Compile o arquivo:
   ```bash
   javac Ex15.java
   ```
4. Execute o programa:
   ```bash
   java Ex15
   ```

## 🧠 Aprendizados

Neste exercício, pude praticar:
- Manipulação de tipos primitivos (`float`, `double`).
- Lógica de decisão para regras de negócio complexas (tabelas de impostos).
- Formatação de saída de dados no console.
